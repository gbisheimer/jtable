The extensions

I developed these extensions for use in some projects I'm working on at our company. I was lucky to find @hikalkan's work for these projects, and in return I wanted to give something back to the community. Hopefully someone will find my extensions usefull as I did with jTable.

My extensions are:

  • Export
  • Filtering
  • Footer
  • Form templates
  • Local DB
  • Master-Child addon
  • Table reload

Master-Child addon

Allows to easily create columns to show child tables displaying related data of the current row, and treats them as a one-to-many relational field. This allows to create automatically the same child table inside the edit or create input forms in order to input all the required data in one place.
